# Minetext
基于 Spring Cloud 开发的插件式博客网站。

# 技术栈
- Spring Cloud Eureka
- Spring Cloud Config
- Spring Cloud Zuul
- Spring Cloud Feign
- Spring Cloud Ribbon
- Spring Boot Security
- Spring Boot Data JPA
- MySQL
